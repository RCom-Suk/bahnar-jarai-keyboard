🌐 Hướng Dẫn Gõ Bàn Phím Bahnar & Jarai trong Keyman

Bàn phím này hỗ trợ các ký tự và dấu thanh đặc biệt trong tiếng Bahnar, Jarai và tiếng Việt.

🔡 Nguyên tắc gõ ký tự đặc biệt

1. Gõ chữ cái đặc biệt trong tiếng Bahnar

Tổ hợp phímKý tựÝ nghĩa
a + w / a + xăNguyên âm ngắn
a + aâNguyên âm đôi
e + xĕe ngắn
e + eêNguyên âm ê
ô + xô̆ô ngắn
u + wưNguyên âm đôi ư
b + bƀÂm “bờ” như tiếng Việt
c + cčÂm vị đặc trưng, như âm “ch”
d + dđÂm d có gạch ngang
n + nñÂm n mũ ~

2. Gõ dấu đặc biệt trong tiếng Jarai

Thêm dấu gạch dưới (\) sau chữ cái:

Ví dụKý tựÝ nghĩa
a + \a̱Dấu gạch dưới tiếng Jarai
ă + \ă̱ă + dấu gạch dưới
â + \â̱â + dấu gạch dưới
e + \e̱e + dấu gạch dưới
ĕ + \ĕ̱e ngắn + dấu gạch dưới
ê + \ê̱ê + dấu gạch dưới
i + \i̱i + dấu gạch dưới
ĭ + \ĭ̱i ngắn + dấu gạch dưới
o + \o̱o + dấu gạch dưới
ô + \ô̱ô + dấu gạch dưới
ơ + \ơ̱ơ + dấu gạch dưới
u + \u̱u + dấu gạch dưới
ư + \ư̱ư + dấu gạch dưới

3. Gõ dấu thanh tiếng Việt

Tổ hợp phímKý tựDấu
a + 1áSắc
a + 2àHuyền
a + 3ảHỏi
a + 4ãNgã
a + 5ạNặng
d + 9đChữ đ

👉 Lưu ý: Khi gõ dấu thanh cho nguyên âm ghép như ă, ê, ô, hãy áp dụng dấu vào chữ ghép (ví dụ: ă + 1 sẽ ra ắ).

📥 Cách cài đặt

  1. Tải & cài ứng dụng Keyman
    👉 Android
    👉 iOS
  2. Mở Keyman, chọn “Install keyboard”
  3. Tìm tới file .KMP vừa tải hoặc nhận qua Zalo, Gmail
  4. Hoàn tất cài đặt, thử gõ trong trình soạn thảo
